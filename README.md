<h1 align="center">Lotrix</h1>
<p align="center">Lo-fi Matrix effect written in C + Raylib.<br>Font made by me.</p>

![](demo.png)
<p align="center">512x288</p>

## Features
- Arguments:
  - `./lotrix [width] [height] [max_amount] [rainbow] [message]`
- Binds:
  - `KEY_SPACE/KEY_P` - Pause/unpause.
  - `KEY_F` - Toggle fullscreen.
